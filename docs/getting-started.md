# Getting Started

Install and run the Voice CLI.

## Install

```bash
./setup.sh install
```

## Verify

```bash
voice --version
voice "Hello world"
```

## Configure API Keys

```bash
voice config set openai_api_key YOUR_KEY
voice config set elevenlabs_api_key YOUR_KEY
voice config set google_api_key YOUR_KEY
```

## Help

```bash
voice --help
voice providers
voice status
```
